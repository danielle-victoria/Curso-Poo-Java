		*Comandos Básicos:
		- Criar um projeto com Maven: mvn archetype:generate -DgroupId=one.digitalinnovation -DartifactId=maven-javadoc-test(nome do projeto) -Darchetype=maven-quick-start -Dinteractive=false
		- Compilar: mvn compile
		- Testar : mvn test
		- Plugins: mvn javadoc:javadoc  --> Gera um site com as informações das classes Java
